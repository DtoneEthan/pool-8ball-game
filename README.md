# 🎱 8球台球 - Pool 8-Ball

一款轻量级网页8球台球游戏，纯HTML/CSS/JS实现，无需任何依赖。

## 🎮 游戏模式

| 模式 | 说明 |
|------|------|
| **基础模式** | 经典8球练习，熟悉规则和操作 |
| **AI对战** | 与智能AI对手对战，AI会寻找最佳击球路径 |
| **双人对战** | 两人轮流击球，适合朋友一起玩 |
| **暴击坏人模式** | 上传讨厌的人的照片，将人脸贴在台球上暴击！支持单人或双人 |

## 🕹️ 操作方式

- **鼠标/触屏**：拖动瞄准，按住蓄力，释放击球
- **空格键**：按住蓄力 / 释放击球
- **R键**：重新开始
- **Esc键**：返回菜单

## 🎯 8球规则

1. 开局后，首次打进实心球(1-7)或花色球(9-15)的玩家确定球型
2. 玩家需要先打完自己的所有球，最后合法打进8号球才能获胜
3. 提前打进8号球判负
4. 白球进袋换对手击球

## 🚀 部署

### 直接打开
直接用浏览器打开 `index.html` 即可。

### 本地服务器
```bash
python3 -m http.server 8000
# 访问 http://localhost:8000
```

### GitHub Pages
1. Fork 本仓库
2. 在 Settings → Pages 中启用 GitHub Pages
3. 选择分支，保存即可

## 🛠️ 技术特点

- **零依赖**：单个HTML文件包含所有代码
- **Canvas渲染**：流畅的60fps物理模拟
- **完整物理引擎**：碰撞检测、摩擦力、反弹、进球检测
- **AI路径规划**：AI会评估球到袋口的最优路径
- **Web Audio音效**：击球、进球、获胜等音效反馈
- **响应式设计**：适配桌面和移动端
- **触屏支持**：完整的移动端触屏操作
